# VihuPlanet World Library v1.0

This is the recommended repository structure for VihuPlanet artwork.

Principles:
- Artwork lives in Git.
- Code assembles and animates artwork.
- Database stores storyteller choices, never master artwork.

Suggested file naming:
background.webp
midground.webp
foreground.webp
details.webp
motion.json
anchors.json
palette.json
README.md
